**2. Create `frontend/src/components/JobMap.jsx`** — read-only map display:
- Import `leaflet/dist/leaflet.css` at top
- Import `L` from `leaflet`, import marker icon PNGs, apply the icon fix (per RESEARCH.md Pitfall 1):
  ```js
  import L from 'leaflet';
  import markerIcon2x from 'leaflet/dist/images/marker-icon-2x.png';
  import markerIcon from 'leaflet/dist/images/marker-icon.png';
  import markerShadow from 'leaflet/dist/images/marker-shadow.png';
  delete L.Icon.Default.prototype._getIconUrl;
  L.Icon.Default.mergeOptions({ iconRetinaUrl: markerIcon2x, iconUrl: markerIcon, shadowUrl: markerShadow });
  ```
- Import `MapContainer, TileLayer, Marker, Popup` from `react-leaflet`
- Props: `{ latitude, longitude, address }`
- Render: `<div className="h-64 w-full rounded-lg overflow-hidden border border-gray-200">` wrapping a `MapContainer` with `center={[latitude, longitude]}`, `zoom={14}`, `style={{ height: '100%', width: '100%' }}`, `scrollWheelZoom={false}`
- TileLayer: `url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"` with OSM copyright attribution
- Single `Marker` at `position={[latitude, longitude]}` with `Popup` showing `{address}` when address is truthy
- Below the map div, if `address` is truthy: `<p className="text-xs text-gray-500 mt-1">{address}</p>`

**3. Create `frontend/src/components/LocationPicker.jsx`** — address search + confirmation map:
- Props: `{ latitude, longitude, address, onChange }` where `onChange({ latitude, longitude, address })` is called when location changes
- State: `query` (search input text), `results` (geocoding results array), `loading` (boolean)
- Import `OpenStreetMapProvider` from `leaflet-geosearch` and create provider instance with `params: { countrycodes: 'ro' }` (per D-01)
- Debounce: use `setTimeout`/`clearTimeout` pattern with 400ms delay (per UI-SPEC). Only fire when `query.length >= 3`
- Render structure:
  1. Label: `<label className="block text-sm font-semibold mb-1">Job Location (optional)</label>`
  2. Search input: `<div className="relative">` containing:
     - `<input type="text" value={query} onChange={...} placeholder="Search address, e.g. Strada Florilor, Cluj" className="w-full border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-600">`
     - When `loading`: show `<svg className="animate-spin h-4 w-4 text-gray-400 absolute right-3 top-2.5">` spinner
  3. Results dropdown: when `results.length > 0`, render `<ul className="absolute z-50 w-full bg-white border border-gray-200 rounded-lg mt-1 shadow-lg max-h-60 overflow-y-auto">` with up to 5 `<li>` items. Each item shows `result.label`. On click: call `onChange({ latitude: result.y, longitude: result.x, address: result.label })`, clear results, set query to result.label
  4. Confirmation map: when `latitude && longitude`, render `<div className="mt-2">` containing `JobMap` with the coordinates. Marker should be draggable — use `useMapEvents` or `Marker` `eventHandlers={{ dragend }}` to update coordinates on drag via `onChange`
  5. Clear link: when `latitude`, show `<button type="button" onClick={() => { onChange({ latitude: null, longitude: null, address: null }); setQuery(''); }} className="text-red-600 text-xs mt-1 hover:underline">Remove location</button>`
- NOTE: For the confirmation map with draggable pin, create a small inline `DraggableMarker` sub-component that uses react-leaflet's `Marker` with `draggable={true}` and `eventHandlers={{ dragend(e) { const ll = e.target.getLatLng(); onChange({ latitude: ll.lat, longitude: ll.lng, address }); } }}`

- Create inline `RadiusSelector` component (or add at bottom of file):
  ```jsx
  function RadiusSelector({ value, onChange }) {
    const options = [10, 20, 50, 100];
    return (
      <div className="flex gap-2">
        {options.map(km => (
          <button
            key={km}
            type="button"
            onClick={() => onChange(km)}
            className={`px-4 py-2 rounded-lg text-sm ${
              value === km
                ? 'bg-blue-600 text-white font-semibold'
                : 'bg-slate-50 text-gray-900 border border-gray-300'
            }`}
          >
            {km} km
          </button>
        ))}
      </div>
    );
  }
  ```
