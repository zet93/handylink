## Interaction Contracts

### Address Search (LocationPicker ŌĆö web and mobile)

- Input debounce: 400ms before Nominatim query fires (Nominatim rate limit: 1 req/s)
- Minimum query length: 3 characters before search triggers
- Suggestion list: shows up to 5 results, each on its own row with full label text
- Selecting a suggestion: populates address text, drops pin on confirmation map, dismisses list
- Confirmation map: 250px height (mobile) / 256px height (web `h-64`), zoom level 14
- Pin is draggable on web (`Marker draggable` prop); not draggable on mobile (address-only input)
- "Clear location" link appears below confirmed address; single tap/click ŌĆö no confirmation dialog (non-destructive, location is optional)
- When no location set: section shows only the search input with placeholder; map is not rendered

### Map Display (JobMap / JobMapMobile ŌĆö read-only)

- Rendered only when job has `latitude` and `longitude`; section is hidden otherwise (no empty state, no placeholder)
- Web: `h-64 w-full rounded-lg overflow-hidden` container with `border border-gray-200`
- Mobile: `height: 250, width: '100%', borderRadius: 10, overflow: 'hidden'` matching existing `borderRadius: 10` on inputs
- Zoom level: 14 (street-level, matches confirmation map)
- OpenStreetMap tile attribution shown (required by OSM license)
- Address text displayed below the map in muted text (13px, weight 400, `palette.muted`) when `address` field is present

### Radius Selector (worker service area)

- Segmented control: 4 inline buttons ŌĆö "10 km", "20 km", "50 km", "100 km"
- Default selected: "20 km" (second option)
- Active button: `palette.accent` background (`#1d4ed8`), white text, `fontWeight: '600'`
- Inactive button: `palette.panel` background (`#f8fafc`), `palette.text` text, `fontWeight: '400'`
- Border: 1px `palette.border` on each button; outer container `borderRadius: 10`
- Height: 44px per button (minimum touch target)
- Web equivalent: 4 `<button>` elements in `flex gap-2`, `px-4 py-2 rounded-lg text-sm`

### Loading States

- Geocoding in progress: show `ActivityIndicator` (mobile) / Tailwind `animate-spin` spinner (web) inline in the search input right side
- Map loading: tiles load progressively ŌĆö no explicit loading state needed (Leaflet/react-native-maps handle this)
- Form submit with location pending: existing `isPending` button disable pattern applies unchanged

## Phase-Specific Notes

### Leaflet Icon Bug (web)
Must apply icon fix before any `Marker` renders. Do it once in `JobMap.jsx` or the Leaflet CSS import location:
```js
delete L.Icon.Default.prototype._getIconUrl;
L.Icon.Default.mergeOptions({ iconRetinaUrl, iconUrl, shadowUrl });
```
Source: RESEARCH.md Pitfall 1.

### MapView Height (mobile)
Always set explicit `height: 250` on `MapView` style ŌĆö never rely on `flex: 1` alone. Source: RESEARCH.md Pitfall 5.

### Romanian Address Diacritics
Display Nominatim's returned label string verbatim rather than the user's raw input. Nominatim may normalize diacritics. Source: RESEARCH.md Pitfall 6.
